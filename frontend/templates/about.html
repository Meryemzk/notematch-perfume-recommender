About the NoteMatch research prototype

NoteMatch is a Django web application created for an undergraduate development project. It explores how mood, scent-note preferences and saved user profiles can support personalised perfume recommendations.

Research purpose

The project investigates whether mood-based and preference-based inputs can make perfume recommendations feel more useful, transparent and user-centred.

Privacy and consent

Users confirm consent before completing the survey. Saved favourites and feedback are used only to improve recommendation quality in this prototype.

Current limitation

The recommendation engine is rule-based. User feedback collected after recommendations can support future machine-learning development.

How the recommendation works

The system compares the user's mood result, preferred notes, three favourite perfumes, gender preference, new/classic style choice and budget against the perfume catalogue. It then ranks suitable perfumes and explains why they were recommended.
